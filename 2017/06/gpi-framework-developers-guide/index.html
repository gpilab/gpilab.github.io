GPI Framework Developer's Guide

Posted by Nicholas Zwart on Tue 06 June 2017

New GPI documentation has been released! Its the moment everyone has been clamoring for. The GPI Framework Developer's Guide is now added to the GPI documentation; hosted on ReadTheDocs. This guide introduces the GPI Framework project, the directory structure, code organization, and uncovers the secrets of mastering a GPI distribution. Its about a 10 minute read that will keep you guessing until the very end! Some have called it "...adequate..."!
