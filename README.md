## Installing

```shell
$ git clone https://github.com/gpilab/gpilab.github.io.git gpi_website
$ cd gpi_website
$ pip install -r requirements.txt
```

## Running

```shell
$ make html
$ make serve
```
The website should now be running on [localhost:8000](http://localhost:8000).

## Publish

```shell
$ make html
$ make github
```
